# 🔐 SecurePass — Password Generator

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=for-the-badge)

A smart command-line password generator that creates strong, customizable passwords and checks their strength — built with pure Python.

---

## ✨ Features

- 🎲 Generate passwords with custom length and character sets
- 💪 Real-time password strength checker with improvement tips
- 📋 Auto-copy to clipboard
- 💾 Save passwords to a local file with labels
- 🛡️ Guarantees at least one of each selected character type

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ETCREALMAN/password-generator.git
cd password-generator

# (Optional) Install clipboard support
pip install pyperclip

# Run
python password_generator.py
```
